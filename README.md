U heeft verschillende opties om deze website gratis online te zetten:

### Optie 1: GitHub Pages (Aanbevolen)

1. **Maak een GitHub account** (gratis op [github.com](https://github.com))

2. **Upload de bestanden naar GitHub:**
   - Klik op "New repository"
   - Geef het een naam (bijv. `noorderbouw-website`)
   - Zet "Public" aan
   - Klik "Create repository"
   - Upload alle bestanden (`index.html`, `styles.css`, `script.js`)

3. **Activeer GitHub Pages:**
   - Ga naar "Settings" in uw repository
   - Scroll naar "Pages" in het linkermenu
   - Bij "Source" selecteer "main" branch
   - Klik "Save"
   - Uw website is nu beschikbaar op: `https://uwgebruikersnaam.github.io/noorderbouw-website`

### Optie 2: Netlify (Zeer Eenvoudig)

1. **Ga naar [netlify.com](https://www.netlify.com)** en maak een gratis account

2. **Drag & Drop:**
   - Log in op Netlify
   - Sleep de map met alle bestanden naar het Netlify dashboard
   - Uw website wordt automatisch gehost!

3. **Of via GitHub:**
   - Verbind uw GitHub account met Netlify
   - Selecteer uw repository
   - Netlify detecteert automatisch de bestanden en host de website

Uw website krijgt een gratis URL zoals: `https://noorderbouw.netlify.app`

### Optie 3: Vercel

1. **Ga naar [vercel.com](https://vercel.com)** en maak een gratis account

2. **Upload bestanden:**
   - Klik op "New Project"
   - Upload de bestanden of verbind met GitHub
   - Vercel deployt automatisch

Uw website krijgt een gratis URL zoals: `https://noorderbouw.vercel.app`

   - Vervang het form action in `index.html`

2. **EmailJS gebruiken** (gratis tier):
   - Maak account op emailjs.com
   - Voeg de JavaScript code toe aan `script.js`
